# Frontend Showcase App

Responsive, animated landing-and-dashboard UI concept (static, deploy-anywhere).

## Highlights
- Distinct typography and visual tone
- Layered gradient + texture background
- Staggered reveal and pulse animations
- Responsive layout (desktop + mobile)

## Run locally
```bash
python -m http.server 8081 --directory src
```
Open `http://localhost:8081`.
